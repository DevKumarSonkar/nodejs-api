Test Case 2 — File Permission Configuration

1. Navigate to home directory and create index.html:
cd ~
touch index.html

2. Add HTML text content:
echo "<html><body><h1>My name is Dev Sonkar</h1></body></html>" > index.html

3. Check default permissions:
ls -l index.html

4. Grant full permissions (read, write, execute) to all users:
chmod 777 index.html

5. Change file owner and group to www-data:
sudo chown www-data:www-data index.html

6. Change file permissions to 755:
sudo chmod 755 index.html

7. Display final permissions and ownership:
ls -l index.html

OUTPUT in Terminal:

dev@pop-os:~$ echo "<html><body><h1>My name is Dev Sonkar</h1></body></html>" > index.html
